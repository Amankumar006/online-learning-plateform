# **App Name**: AdaptEd AI

## Core Features:

- Dashboard: Personalized dashboard showing progress and lessons.
- Lessons: Interactive lessons that supports text, images, and videos.
- Adaptive Exercises: Adaptive exercises that adjust difficulty in real-time.
- Progress Tracking: Visual progress tracking and mastery charts.
- Buddy AI: AI-powered study buddy to incorporate knowledge to generate different types of conversation (using Genkit as a tool)
- Authentication: Secure authentication with Firebase Auth

## Style Guidelines:

- Primary color: A calming blue (#64B5F6) to promote focus and trust in the platform. Chosen for its association with knowledge and stability, but without being clich√©.
- Background color: A very light blue (#E3F2FD) as a subtle backdrop.
- Accent color: A warm orange (#FFB74D) for calls to action and important notifications. Provides a stimulating, encouraging pop.
- Body and headline font: 'Inter' (sans-serif) for clear and modern readability. This font provides a neutral, machined appearance, ideal for creating the perception of objective information.
- Consistent and clear icons from shadcn/ui to guide navigation. Favor icons which are simple line drawings over more ornamental alternatives.
- Clean, modular layout to emphasize key information and improve the user experience.
- Subtle transitions for exercise difficulty adjustments to show AI adaptivity.
